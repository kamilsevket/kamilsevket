### What I'm working on

- A **Swift Package SDK** powering a small family of hyper-casual iOS puzzle games — shared engine, per-game twists pluggable through a single protocol
- The first game built on that SDK, with Game Center leaderboards and full UI test coverage
- Personal experiments around modern Swift concurrency, SpriteKit + SwiftUI interop, and small developer tooling

### A bit more about me

- I prefer native Apple tech first — SwiftUI before web wrappers, SwiftData before sidecar databases, GameKit before custom social stacks
- I like architectures with one write lane, value-typed snapshots, and protocols as the only extension point
- Outside of iOS I dabble with AI coding agents and small developer tooling
